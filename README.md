# Mortgage Refinance Calculator

## Overview
This mortgage refinance calculator helps users determine potential savings from refinancing their current mortgage. By inputting key loan details, users can see how much they could save by switching to a new mortgage with different terms.

## Features
- Calculate monthly payments for existing and new loans
- Determine total interest savings over the life of the loan
- Compare different loan scenarios

## Usage
1. Input your current mortgage details:
   - Current loan amount
   - Current interest rate
   - Remaining loan term
2. Input the details for the new loan:
   - New loan amount
   - New interest rate
   - New loan term
3. The calculator will display your current monthly payment, new monthly payment, and total savings over the loan term.

## Installation
To install the necessary dependencies, run:
```
npm install
```
